Vent Parameters



x

  Type: number
  Related: y, contentX, contentY

The position of the vent on the X-axis.



y

  Type: number
  Related: x, contentX, contentY

The position of the vent on the Y-axis.



contentX

  Type: number
  Related: x, y, contentY

The position of the vent's content group on the X-axis.



contentY

  Type: number
  Related: x, y, contentX

The position of the vent's content group on the Y-axis.



build

  Type: function
  Related: (none)

A function that builds and returns a display object that CBEffects will use as a particle.



propertyTable

  Type: table
  Related: (none)

A table of properties that are transferred to each particle on creation.



scale

  Type: number
  Related: scaleX, scaleY

The scale of the vent. Used as default for scaleX and scaleY.



scaleX

  Type: number
  Related: scale, scaleY

The scale of the vent on the X-axis. Defaults to scale.



scaleY

  Type: number
  Related: scale, scaleX

The scale of the vent on the Y-axis. Defaults to scale.



parentGroup

  Type: Corona display object
  Related: (none)

The group that the vent's content will be inserted into.



isActive

  Type: Boolean
  Related: (none)

Whether this vent accepts startMaster() and emitMaster() commands in a VentGroup.



title

  Type: string
  Related: (none)

The access-ID of the vent in a VentGroup. Used in VentGroup commands that apply only to specific vents.



rotateTowardVel

  Type: Boolean
  Related: towardVelOffset

Whether particles rotate to face the direction they are travelling.



towardVelOffset

  Type: number
  Related: rotateTowardVel

An offset by which particles are rotated toward their direction if rotateTowardVel is enabled.



color

  Type: table
  Related: cycleColor, curColor, colorIncr

A table containing inner tables that specify the color of particles. The inner tables must be in the form of {r, g, b}.



cycleColor

  Type: Boolean
  Related: color, curColor, colorIncr

If set to true, particle colors will be cycled through in order. If set to false, particles will be colored randomly from the color table.



curColor

  Type: number
  Related: color, cycleColor, colorIncr

The color color cycling starts with if cycleColor is true.



colorIncr

  Type: number
  Related: color, cycleColor, curColor

The value by which the current color is incremented if cycleColor is true.



emitDelay

  Type: number
  Related: perEmit, emissionNum

The delay between particle emission sequences.



perEmit

  Type: number
  Related: emitDelay, emissionNum

The number of particles emitted for each emission sequence.



emissionNum

  Type: number
  Related: emitDelay, perEmit

The number of emission sequences to toggle when vent.start() is called.



startAlpha

  Type: number
  Related: lifeAlpha, endAlpha

The alpha particles are when they are created.



lifeAlpha

  Type: number
  Related: startAlpha, endAlpha

The alpha particles are during their life span.



endAlpha

  Type: number
  Related: startAlpha, lifeAlpha

The alpha particles are when they are destroyed.



inTime

  Type: number
  Related: lifeTime, outTime

The duration of the transition from startAlpha to lifeAlpha.



lifeTime

  Type: number
  Related: inTime, outTime

The duration of the particle's life span, when no alpha changes are taking place and the particle's alpha is lifeAlpha.



outTime

  Type: number
  Related: inTime, lifeTime

The duration of the transition from lifeAlpha to endAlpha.



onCreation

  Type: function
  Related: onCreationTime, onDeath, onUpdate

A function called when a particle is created. The particle, the vent, and the vent's content group are passed as arguments.



onCreationTime

  Type: string
  Related: onCreation

When during the particle's creation sequence to call the onCreation function.

Supported Values:

  • afterBuild: Calls the function directly after the particle is built. The particle has only the CBEffects particle core methods and values.
  • afterVel: Calls the function once the particle's velocity has been set.
  • afterColor: Calls the function once the particle has been colored and has the .colorChange() function available.
  • afterPosition: Calls the function once the particle has been positioned.



onDeath

  Type: function
  Related: onCreation, onUpdate

A function called when the particle's life span is over and it is destroyed. The particle, the vent, and the vent's content group are passed as arguments.



onUpdate

  Type: function
  Related: onCreation, onDeath

A function called each frame for each particle that is updated. The particle, the vent, and the vent's content group are passed as arguments.



onVentInit

  Type: function
  Related: (none)

A function called when a vent is created using the parameters. The vent is passed as an argument. This is a good place to set up extra vent properties or custom values.



onEmitBegin

  Type: function
  Related: onEmitEnd

A function called directly before an emission sequence begins.



onEmitEnd

  Type: function
  Related: onEmitBegin

A function called directly after an emission sequence ends.



positionType

  Type: string
  Related: radius, xRadius, yRadius, innerRadius, innerXRadius, innerYRadius, point1, point2, lineDensity, rectLeft, rectTop, rectWidth, rectHeight, pointList, cyclePoint, curPoint, x, y

Specifies the way particles are positioned.

Supported Values:

  • inRadius: Makes particles appear inside an elliptical shape. To customize, see radius, xRadius, yRadius, innerRadius, innerXRadius, and innerYRadius.
  • alongLine: Makes particles appear along a line. To customize, see point1, point2, and lineDensity.
  • inRect: Makes particles appear inside of a rectangle. To customize, see rectLeft, rectTop, rectWidth, and rectHeight.
  • atPoint: Makes particles appear at a single (x, y) point. To customize, see x and y.
  • fromPointList: Makes particles appear on multiple points from the pointList. To customize, see pointList, and cyclePoint.
  • custom function: Position particles using your own custom function. It must return two numbers, x and y, respectively. To use this capability, make this parameter, positionType, a function.



radius

  Type: number
  Related: positionType, xRadius, yRadius, innerRadius, innerXRadius, innerYRadius

The master size of the ellipse particles will appear in if positionType is inRadius. Acts as a default for xRadius and yRadius.



xRadius

  Type: number
  Related: positionType, radius, yRadius, innerRadius, innerXRadius, innerYRadius

The half-width of the ellipse particles will appear in if positionType is inRadius. Defaults to radius.



yRadius

  Type: number
  Related: positionType, radius, xRadius, innerRadius, innerXRadius, innerYRadius

The half-height of the ellipse particles will appear in if positionType is inRadius. Defaults to radius.



innerRadius

  Type: number
  Related: positionType, radius, xRadius, yRadius, innerXRadius, innerYRadius

The master size of the ellipse particles will be blocked from appearing in if positionType is inRadius. Acts as a default for innerXRadius and innerYRadius.



innerXRadius

  Type: number
  Related: positionType, radius, xRadius, yRadius, innerRadius, innerYRadius

The half-width of the ellipse particles will be blocked from appearing in if positionType is inRadius. Defaults to innerRadius.



innerYRadius

  Type: number
  Related: positionType, radius, xRadius, yRadius, innerRadius, innerXRadius

The half-height of the ellipse particles will be blocked from appearing in if positionType is inRadius. Defaults to innerRadius.



point1

  Type: table
  Related: positionType, point2, lineDensity

The (x, y) location of the first point of the line particles will appear on if positionType is alongLine. Provided in the form of {x, y}.



point2

  Type: table
  Related: positionType, point1, lineDensity

The (x, y) location of the second point of the line particles will appear on if positionType is alongLine. Provided in the form of {x, y}.



lineDensity

  Type: table
  Related: positionType, point1, point2

The number of points to calculate along the line if positionType is alongLine. If set to the string total, will make an unbroken line.



rectLeft

  Type: number
  Related: positionType, rectTop, rectWidth, rectHeight

The left side of the rectangle particles appear inside if positionType is inRect.



rectTop

  Type: number
  Related: positionType, rectLeft, rectWidth, rectHeight

The top side of the rectangle particles appear inside if positionType is inRect.



rectWidth

  Type: number
  Related: positionType, rectLeft, rectTop, rectHeight

The width of the rectangle particles appear inside if positionType is inRect.



rectHeight

  Type: number
  Related: positionType, rectLeft, rectTop, rectWidth

The height of the rectangle particles appear inside if positionType is inRect.



pointList

  Type: table
  Related: positionType, cyclePoint, pointIncr, curPoint

The list of points particles will appear in if positionType is fromPointList. Provided in the form of {{x, y}, {x, y}, {x, y}, ... }.



cyclePoint

  Type: Boolean
  Related: positionType, pointList, pointIncr, curPoint

Whether points are chosen in order or randomly from the pointTable if positionType is fromPointList. If set to true, points are chosen in order. If false, they are picked randomly.



pointIncr

  Type: Boolean
  Related: positionType, pointList, cyclePoint, curPoint

The value by which the current point is incremented if cyclePoint is true.



curPoint

  Type: Boolean
  Related: positionType, pointList, pointIncr, curPoint

The point point cycling starts with if positionType is fromPointList.
